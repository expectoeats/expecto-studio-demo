# Hero Video

Place your hero video file here as `hero.mp4`.

Requirements:
- Format: MP4 (H.264)
- Resolution: 1920x1080 or higher
- Duration: 15-30 seconds (looping)
- File size: Keep under 15MB for performance

The video will be:
- Autoplay, muted, looped
- Filtered: brightness(0.45) saturate(0.8)
- Parallax scrolled with GSAP ScrollTrigger
